## Задания:

У Вас есть приложение products, там одна views, один шаблон index.html
и две модели в models.
В таблицах уже есть данные, осталось их вывести.

Необходимо по ниже перечисленному списку передать переменные с объектами models в context, 
и вывести в шаблоне index.html. Пример одной переменной уже есть во views.

1. Вывести количетство всех продуктов
2. Вывести только ягоды
3. Вывести фрукты и овощи
4. Вывести продукты количество которых больше 10
5. Вывести продукты количество которых меньше 10
6. Вывести только импортные продукты
7. Вывести только местные ягоды
8. Вывести просроченные продукты
9. Вывести яблоки
10.Вывести все кроме яблок
11.Вывести все фрукты и отсортировать по имени  # sort_by()
12.Вывести все продукт # реализованно
